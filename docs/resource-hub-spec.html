5. Behavior

Filtering

  • Topic dropdown, Type dropdown, and Search input are AND'd together
  • Search matches case-insensitively against title + description
  • When any filter is active, a results banner appears showing count + active filter chip(s)
  • Clearing filters resets all three inputs

Pagination

  • 9 cards per page
  • Pagination operates on the filtered subset
  • Resets to page 1 when any filter changes
  • Shows "Showing X–Y of Z resources" count
  • Page numbers with ellipsis for large sets

Card colors

  • The thin color strip at the top of each card auto-cycles by index
  • Cycle order: forest → meadow → heritage → tidewater → repeat
  • No color field in JSON — derived at render time

Links

  • href starting with / → Gatsby <Link> (internal navigation)
  • All other href → <a target="_blank" rel="noopener noreferrer">
